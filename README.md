# wedding-management-
A comprehensive platform designed to simplify wedding planning by consolidating all essential services in one place. Users can select their cultural preferences, choose an event date, and book venues, catering services, DJs, decorators, and photographers. The system also supports mock credit card payments and allows users to print receipts for their bookings. It aims to streamline the entire wedding planning process, making it efficient and hassle-free for both couples and wedding planners.

## How to Run
First Time:

1. In Your terminal, type *git clone ...*
2. navigate to wedding-management- *cd wedding-management*
3. Run *npm install* to install backend dependancies.
4. Run *cd frontend* into the frontend, then *npm install*
5. Navigate back to backend *cd ..*
6. Run *npm run dev* to run the code locally, then access though **localhost:3000/**on local server.
